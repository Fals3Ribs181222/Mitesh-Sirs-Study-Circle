# Communication and Materials

The system provides tools for disseminating information and sharing study resources.

## Communication (Announcements)
The announcements feature serves as an internal noticeboard for the tuition center.

1.  **Posting:** Teachers can create and post announcements. These can be general notices or targeted to specific grades. Announcements are stored in the `announcements` table, recording the title, message, target grade, and the teacher who posted it.
2.  **Viewing:** Announcements are publicly viewable by everyone in the system. Students see relevant announcements on their dashboards.

## Study Materials (Files)
The files feature allows for the secure sharing of educational documents.

1.  **Storage:** The system utilizes Supabase Storage, specifically a bucket named `materials`, to host the actual file content.
2.  **Metadata Tracking:** When a teacher uploads a file to the `materials` bucket, a corresponding record is created in the `files` table within the PostgreSQL database. This record stores metadata such as the file title, target grade, subject, the public URL of the uploaded file (`file_url`), and the identity of the uploading teacher.
3.  **Access:** Similar to announcements, files metadata is publicly viewable. Students can browse and access relevant shared materials for their grade and subjects through their interface. Files are accessed via the public URLs generated by the Supabase Storage service.
